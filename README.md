# K-POP Media Localization & Integration Platform
這是一個獨立開發的影音整合平台，旨在優化韓語節目的在地化觀影體驗。

### 🚀 技術棧 (Tech Stack)
- 前端：HTML5, CSS3, JavaScript (Vanilla JS)
- 後端：Firebase (Real-time Database)
- 部署：GitHub Pages

### ✨ 核心功能 (Key Features)
- **多樣化佈局**：設計了三種不同的網頁版面與卡片式作品呈現。
- **影音優化**：支持跨裝置全螢幕播放，並具備自定義字幕字體大小功能。
- **即時互動**：利用 Firebase 實作留言牆與即時表情符號反饋。
- **搜尋功能**：開發全文關鍵字搜尋系統，提升內容檢索效率。
- **安全性設計(Security Architecture)**：實施了 Firebase Security Rules 進行細粒度的權限控管，確保數據僅能以預期格式（數值驗證）寫入，並有效防止惡意刪除與非授權存取。
